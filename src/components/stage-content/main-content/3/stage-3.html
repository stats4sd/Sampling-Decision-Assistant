Decisions about the target population

Table 4 “Standard categories of analysis” in the UNHCR Needs Assessment Handbook provides a useful description of criteria for the analytical framework of assessments. This framework is useful to specify the target population of the survey. In most cases, these populations are constructs that are determined by the objectives for which the study is being set up. Getting the boundaries of the population right, means that it would be possible to manage potential biases in the study.
